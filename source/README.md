# Pirith Language Flags

This resource pack adds flag icons for the language selection GUI in `PirithMain`.

The plugin uses `PAPER` with these custom model data values:

- `10001` English
- `10002` Nederlands
- `10003` Portugues
- `10004` Espanol
- `10005` Deutsch
- `10006` Russkiy
- `10007` Francais

Zip the contents of this folder, not the folder itself, when installing it as a server resource pack.
